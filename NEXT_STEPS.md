## 🐛 Troubleshooting

### Dashboard not deploying?
- Check: https://github.com/MastaTrill/Aetheron_platform/actions
- Look for red ❌ in workflow runs
- Make sure you selected "GitHub Actions" in Pages settings

### Price data not showing?
- Open console (F12) - check for errors
- DexScreener might be rate-limiting (rare)
- Pair address might be wrong (unlikely)

### Can't access dashboard?
- Wait 3-5 minutes after first enabling Pages
- Try hard refresh (Ctrl+F5)
- Check if repository is public
